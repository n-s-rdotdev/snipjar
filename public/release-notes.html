v1.0.0

Release notes

The first release of Snipjar focuses on a fast macOS launcher for storing, finding, copying, and pasting local snippets without leaving the keyboard.

Highlights

  • Launch Snipjar from anywhere with Control + Option + Space and search recent snippets immediately.
  • Create, edit, delete, and tag snippets in a compact desktop workflow.
  • Copy or paste the selected snippet back into the previously active app with a macOS-first flow.

Shortcut keys

  • Open or hide the launcher Control+Option+Space
  • Paste the selected snippet into the previous app Return
  • Create a new snippet Command+N
  • Edit the selected snippet Command+E
  • Delete the selected snippet Command+Backspace
  • Move through search results UpDown

Behavior notes

  • Snippets are stored locally on-device in SQLite.
  • Automatic paste may require Accessibility permission on macOS.
  • The menu bar tray now includes Show, Release notes, and Quit for quick access.
